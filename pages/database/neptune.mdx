# Amazon Neptune service

---
title: Amazon Neptune
---

## Definition

Amazon Neptune is a fully managed graph database service provided by Amazon Web Services (AWS). It is designed to store and query highly connected data, making it well-suited for applications that require complex relationships and dependencies.

## Overview

**Key Features:**
1. **Graph Database Engine:** Supports popular graph models like Property Graph and RDF.
2. **Fully Managed Service:** AWS handles hardware provisioning, setup, configuration, patching, and backups.
3. **High Performance:** Utilizes a purpose-built, high-performance graph engine for efficient queries.
4. **Scalability:** Automatically replicates data across multiple Availability Zones for high availability and supports read replicas.
5. **Security:** Provides network isolation using Amazon VPC, encryption at rest, and encryption in transit.
6. **Compatibility:** Works with graph query languages such as Apache TinkerPop Gremlin and SPARQL.

## Use Cases

Amazon Neptune is suitable for a variety of use cases, including:
- Social networking applications
- Fraud detection
- Recommendation engines
- Knowledge graphs
